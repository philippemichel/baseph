# Données (fictives & partielles) sur les escarres en réanimation

Données (fictives & partielles) sur les escarres en réanimation

## Usage

``` r
patients
```

## Format

A data frame with 50 rows and 8 variables:

- sexe:

  Sexe

- age:

  Age du patient

- admission:

  Mode d'admission

- lieudevie1:

  Lieu de vie avant

- igs2:

  Score IGS2

- typemaladie:

  médical ou chirurgical

- sejourrea:

  Durée de séjour en réanimation (j)

- escarre:

  Présence d'une escarre (donnée fictive)
